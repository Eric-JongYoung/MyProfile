한줄소개

IT개발 스터디 모집 및 개발자 커뮤니티

✅ 상세내용

Kosta 에서 교육을 받으면서 마지막으로 진행하였던 저의 3번째 IT 프로젝트 이며, Spring Boot 기반으로 제작한 프로젝트 입니다.
교육때 배운 Jsp&Servelt 기반이 아닌, Thymeleaf 를 front로 사용하였으며, AWS를 이용하여 배포까지 완료한 프로젝트 입니다.

IT 업계의 화두 중 하나는 소위 ‘개발자 구하기 전쟁’이다. 알파고 이후 인공지능의 급부상과 카카오 등 플랫폼 기업의 약진, 그리고 암호화폐와 NFT으로 대표되는 블록체인 분야의 기대감 확대와 이를 아우르는 메타버스의 미래에 대한 기대 등으로 IT, 특히 소프트웨어 분야는 다시 미래 핵심 경쟁력으로 부각되고 있다. 기업에서 요구하는 개발자의 역량은 날로 높아지고 있으며, 이에 발맞춰 개발자들은 자기개발의 필요성을 느끼고 있다. 물론 혼자서 공부할 수 있다. 하지만, 협업이 중요시되는 현시대에 맞춰 Jambo는 IT 스터디원을 구할 수 있도록 하는 웹사이트를 목표로 하고있으며, 다양한 IT 정보를 개발자들끼리 공유할 수 있는 커뮤니티도 제공한다.

🛠️ 개발환경

  • Spring Boot 2.7.6, Spring Security
  • Java 11, Gradle
  • JPA, Oracle 19c
  • Thymeleaf
  • SummerNote API, SSE(Server-Sent-Event)
  • AWS
  • Mac OS Ventura 13.0.1
  • intelliJ IDEA
  • Notion, Discord

👾 담당 기능

  • [회원가입 기능 구현]
    • - Spring Security 적용
    • - 회원가입시 아이디 중복체크를 AJAX로 기능 구현
  • [스터디 게시판 기능 구현]
    • - 스터디 게시판 글 작성시 SummerNote API를 이용하여 게시글 꾸미기 기능 구현
    • - 회원이 스터디 게시글 마감 기능 구현
    • - 마감된 스더터 게시글이 게시글 리스트에서 상태에 따라 front 에서 다르게 보이도록 기능 구현
  • [일반 게시판 기능 구현]
    • - 게시글 등록, 수정, 삭제 기능 구현
    • - 게시글 좋아요 기능 구현 (회원만 가능하며, 회원은 같은 글에 중복으로 좋아요 불가)
    • - 게시글 신고하기 기능 구현 (회원만 가능하며, 회원은 같은 글에 중복으로 신고 불가)
    • - 회원만 게시글 작성 가능하도록 Srping Security 적용
  • [관리자 페이지 기능구현]
    • - 관리자 메인페이지에서 회원이 작성한 게시글, 회원가입 수, 회원 정보, 신고된 게시글 수 , 스터디 게시글 수 조회 기능 구현
    • - 회원이 신고한 게시글 신고 사유 조회 기능 구현
    • - 회원이 신고한 게시글을 게시판에서 숨기기, 활성화 기능 구현
    • - Spring Security 를 이용하여 관리자 계정 생성 기능 구현
    • - 관리자페이지에서 공지사항, 소작모, 행사 게시판 글쓰기
  • [AWS를 통한 배포]
    • - AWS EC2 인스턴스 Amazon Linux 환경
    • - ssh를 이용하여 인스턴스 연결하여 프로제트 build & run

💎 주요 기능 소개

[메인페이지]

- 사이트의 전반적인 내용 소개

[스터디 게시판]

- 스터디 게시판 글 작성시 SummerNote API를 이용하여 게시글 꾸미기 기능 구현

- 회원이 스터디 게시글 마감 기능 구현

[일반게시판 - 좋아요기능]

- 좋아요 기능 구현 (회원만 가능하며, 게시글당 하나의 좋아요만 기능구현)

[일반 게시판 - 신고하기]

- 신고 기능 구현 (회원만 가능하며, 게시글당 하나의 좋아요만 기능구현)

- 관리자가 해당 신고 내용을 관리자페이지에서 조회 가능하며, 게시글 숨기기를 관리자가 할 수 있다.

[아이콘샵]

- 회원은 게시글 작성 및 댓글 작성, 또한 자신이 작성한 게시글이 좋아요 받을시 포인트를 받아서 아이콘샵에서 아이콘을 구매가능

- 아이콘 등록은 관리자페이지에서 관리자가 등록 할 수 있다.

  • - 해당 프로젝트를 통하여 MVC 구조와 Spring Boot & Spring Security 의 구조를 이해하게 됨.
