✅ 상세내용

저의 첫번째 it 프로젝트 이며, Java Console 기반으로 제작한 프로젝트 입니다.

영화티켓 16,000원시대! 소비자들은 재미없는 영화를 보고싶어하지않는다!
다른 소비자들의 영화리뷰를 '리무버'를 통해 미리 확인하며 자신이 볼영화를 선택할수있다.
또한 자신이 남긴리뷰를 통해서 다른소비자에게 도움을준다!
이를통해서 영화관계자는 소비자들의별점, 리뷰를 확인함으로써, 영화관계자는 소비자의니즈를 파악할수있다!

🛠️ 개발환경

  • Java 11
  • Oracle 19c
  • Mac OS Monterey, Windows 10
  • intelliJ IDEA

👾 담당 기능

역할 : 팀장
  • 영화 검색
    • 관객수 db기반 top5 기능 구현
  • 관리자페이지
    • 관리자 로그인, 회원가입 기능 구현
    • 회원정보 조회 기능 구현
    • 관리자 공지사항 작성 기능 구현
  • 취향저격 콘텐츠
    • 회원이 회원가입시 선택한 취향 장르를 바탕으로 db의 관객수가 많은 영화를 회원에게 취향 저격 영화로 추천 기능 구현
  • Oracle Cloud 연동
    • Local Oracle이 아닌, Oracle Cloud 의 ATP를 사용하여 프로젝트와 JDBC를 통하여 연동을 진행하였다.

💡 깨달은점

  • 처음으로 Git을 사용해 봄으로써 git 에대한 개념이 부족해서 merge에 대한 지식이 부족하였으나, 자주 사용하면서 기능을 익힌뒤 git 사용법을 어느정도 깨닫게 되었다.
  • 처음으로 it 프로젝트를 진행해보았으며, 최대한 MVC 구조를 지키면서 코드를 작성하려고 노력하였다. 이를 통해서 MVC 구조에 대하여 익혔다.
  • 초반에 Oracle Cloud 의 Oracle 과 Local Oracle 연동에필요한 라이브러리의 차이를 모르고 연동에 애를 먹었다. 하지만, 이를 해결하면서 스스로 문제 해결 극복 능력을 이루었다.

🔥 보완해야할 점

  • Java Console 로 이루어지다보니, 프로젝트 내용이 한눈에 보이지 않았다. UI에 조금더 신경 쓸 수 있도록 보완의 필요성.
  • 영화 정보를 가져올때, 외부에서 가져온 정보를 parsing해서 DB에 직접 넣었으나, 다음에는 API 를 이용해서 자동적으로 DB에 최신 영화가 입력될 수 있도록 해야할 필요성.
  • Git Commit 을 진행할때, Commit Message 를 무작위로 작성하여서 내가 어디서 무엇을 Commit 했는지 이해하기 힘들었다. 이를 다음에는 팀원들과 맞춰서 Message 를 일관성 있게 작성해야하는 필요성.
